# TGBot-interpreter
Link to the bot in the telegram: @InterpretiveBot

Bot functionality description:
After starting the bot, you can choose from three translation styles:
1. Regular (standard, neutral);
2. Slang (casual, informal, colloquial);
3. Professional (formal, business‑like).

You can also select the target language: Russian or English.
The bot includes input validation to prevent errors: you can only choose from the predefined styles and languages (no custom entries allowed).

Additionally, you can:
1. Select a language (Russian or English);
2. Choose a style (other than Regular);

Enter your own sentence in the selected language.

The bot will then rephrase your sentence in the chosen style—without changing the language.
